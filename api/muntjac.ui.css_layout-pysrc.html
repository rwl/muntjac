 30 -class CssLayout(AbstractLayout, ILayoutClickNotifier): 

 31      """CssLayout is a layout component that can be used in browser environment 
 32      only. It simply renders components and their captions into a same div 
 33      element. Component layout can then be adjusted with css. 
 34   
 35      In comparison to L{HorizontalLayout} and L{VerticalLayout} 
 36   
 37        - rather similar server side api 
 38        - no spacing, alignment or expand ratios 
 39        - much simpler DOM that can be styled by skilled web developer 
 40        - no abstraction of browser differences (developer must ensure that 
 41          the result works properly on each browser) 
 42        - different kind of handling for relative sizes (that are set from 
 43          server side) (*) 
 44        - noticeably faster rendering time in some situations as we rely more 
 45          on the browser's rendering engine. 
 46   
 47      With L{CustomLayout} one can often achieve similar results (good 
 48      looking layouts with web technologies), but with CustomLayout developer 
 49      needs to work with fixed templates. 
 50   
 51      By extending CssLayout one can also inject some css rules straight to 
 52      child components using L{getCss}. 
 53   
 54      (*) Relative sizes (set from server side) are treated bit differently than 
 55      in other layouts in Muntjac. In cssLayout the size is calculated relatively 
 56      to CSS layouts content area which is pretty much as in html and css. In 
 57      other layouts the size of component is calculated relatively to the "slot" 
 58      given by layout. 
 59   
 60      Also note that client side framework in Muntjac modifies inline style 
 61      properties width and height. This happens on each update to component. If 
 62      one wants to set component sizes with CSS, component must have undefined 
 63      size on server side (which is not the default for all components) and the 
 64      size must be defined with class styles - not by directly injecting width 
 65      and height. 
 66      """
